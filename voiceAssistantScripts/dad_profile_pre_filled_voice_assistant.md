# Context Packs — Outlines (RAG‑friendly)
> Each pack is a small, focused set of notes/files the assistant can search. Keep titles clear, add dates, and include 2–3 keywords.

## How to Use Packs
- **Structure:** One folder per pack; each item is a short note/file named `YYYY‑MM‑DD — Title`.  
- **Include in each item:** 1–2 line summary, details, and “What to do next.”  
- **Tag keywords:** 2–3 terms Dad would say (e.g., “WhatsApp”, “answer call”).  
- **Update cadence:** Note how often you’ll refresh (e.g., after each visit).

---

**Purpose:** Quick answers about today (date, day, weather, top 3 plans).  
**Typical items:** “Today overview”, “This week’s highlights”.

**Save rules:** Auto‑generate each morning; overwrite or archive by date.

# Ideas & Use Cases (expanded)
**Communication & Contacts**  
- Call or message family by name; read recent messages on request.  
- “Who is <name>?” → relation + last interaction.  
- Birthday/anniversary reminders and simple messages.  
- “Summarize my last 3 chats with Michael” → 3 bullets each.

**Orientation & Day Structure**  
- Morning briefing (day/date, weather, top 3).  
- Evening wrap‑up (wins, tomorrow’s first step).  
- Gentle check‑ins if there’s been no contact by a set time.  
- “What’s my plan between 2 and 4?” (fills the gap with options).

**Notes, Lists, and Retrieval**  
- Voice‑capture notes tagged to people/places (“Garden: fix gate latch”).  
- Shopping lists by store; chores by room.  
- “What did we decide about the plumber?”  
- “Find notes that mention ‘printer’ in the last month.”

**Appointments & Admin**  
- Dictate/reschedule messages; capture call outcomes in 1 line.  
- Summarize photographed letters/bills + suggested next steps.  
- “Make a polite complaint about missed bin collection” (template).

**Tech Help (A05 & headset)**  
- Walkthroughs: answer WhatsApp, reconnect Bluetooth, Wi‑Fi quick fix.  
- Practice mode for any of the above.  
- “What’s my battery? Remind me to charge if under 20%.”

**Confidence & Initiation**  
- One‑minute start cues for walks/tasks (count‑in, posture, first action).  
- 3‑minute focus timer + first step read‑out.  
- “Talk me through starting a phone call.”

**Household Logistics**  
- Bill due‑date reminders and provider contact recall.  
- “Where are the spare keys?” (if noted).  
- “Start ‘Garden jobs’ and add weeding + hose repair.”

**Social & Engagement**  
- Suggest a short story, a joke, or a memory prompt.  
- “Give me a talking point for Sarah before I call.”  
- “What did Sarah say last time we spoke?”

**Safety & Escalation**  
- Trigger phrases to contact Michael; optional place‑label hint (no precise location sharing).  
- Missed check‑in → offers to call you.  
- “If I say ‘I’m lost’, read the home address label” (store safely).

**Travel**  
- “What time should I leave for the clinic?” (with buffer).  
- Out‑the‑door checklist; return check‑in.

**Future (when sensors arrive)**  
- Time‑based gait prompts; correlate mood with activity summaries; adaptive coaching.
